****************Aplikacja Top 20 Song Chart*******************
Wymagania:
-Zainstalowany Python w wersji 3.x
-Zainstalowany pakiet Django w wersji 3.2+

Aby uruchomić aplikację należy otworzyć folder zawierający aplikację w wierszu polecen i wpisać komendę:
$ python manage.py runserver
Następnie należy otworzyć przeglądarkę i wpisać w pasku adresu:
localhost:8000
